El HB030 es el identificador de hogar del conjunto de th_2025 . Ahora tengo que buscar en el diccionario de variables cual es el código de identificación para poder unir los 4 conjuntos de datos.

Después de buscarlas, tengo esto:

  • HB030 → identificador del hogar (en fichero de hogar)
  • DB030 → identificador del hogar (en fichero de composición)
  • PB030 → identificador de persona = HB030 + nº de orden de persona
  • RB030 → identificador de persona = DB030 + nº de orden de persona

  • DB030 y HB030 → son números simples: 1, 2, 3, 4…

  • PB030 y RB030 → son el número del hogar + nº de orden de persona: 101, 201, 202, 301…

Es decir, en PB030 = 202 el hogar es 2 y la persona es la 02 de ese hogar. 1.1.2 2. Creación del conjunto de datos final

Una vez tenemos todos los ficheros con los mismos valores procedemos a juntarlos para tener un unico dataset. Primero voy a unir el conjunto de composición de hogar con el de hogar, luego con el de persona y luego con el de renta. Voy a usar un left_join para conservar la parte izquierda de las tablas que es donde esta el identificador de hogar:

# Unir hogar y composición del hogar
df_2025 <- td_2025 |> 
  left_join(th_2025, by = "id_hogar")
# Unir con persona
df_2025 <- df_2025 |> 
  left_join(tp_2025, by = "id_hogar")

Para unir con la renta, en vez de unir por el id_hogar tengo que unirlo por el identificador de persona porque la renta es a nivel individual no a nivel de hogar:

# Unir con renta por identificador de persona
df_2025 <- df_2025 |> 
  left_join(tr_2025, by = c("PB030" = "RB030"))
